Bedienung Funktion MAN:


  Hauptfunktion MAN (Manuell).
  Die Objektcodes werden in einem Editfenster dargestellt;
  können mit einem Editor erzeugt / modifiziert / gelöscht werden.
  Die Darstellung erfolgt simultan im Grafikfenster.


Aktivieren der Funktion MAN:
   Die Checkbox MAN selektieren.


____________________________________________________________________________
Erzeugen von Objekten:
  Eine Codezeile im Editfenster eingeben,
    Abschluss mit Returntaste oder rechter Maustaste.
    Die Zeile wird bearbeitet, das Objekt wird dargestellt.

  Eine Codezeile für ein geometrisches Objekt besteht aus:
    Objektname = Objektparameter. Beispiel für einen Punkt:
    P10 = P(100 100 0)
    CodeDefinitionen: siehe gCAD3D-Codes

  Neue Objektnamen können mit der Alt-Taste und dem Objekt-Character
    erzeugt werden; Alt-P erzeugt also den Text "P20=".
    Die aktuelle Zeile muss leer sein.
    Weiters: Alt-L Line, Alt-C Kreis, Alt-S Kurve usw.

  Zugehörige geometrische Objekte (Punkte, Konturen) können selektiert werden.
    Der Objektname des selektierten Objekts wird im Editfenster zugefügt.
    Kein Objekt ("indicate"): es wird die Cursorposition
      auf der Konstruktionsebene als Punktobjekt zugefügt.
    
    Parametrische Definition für Punkte werden nur im Modus CAD erzeugt.
   
   
____________________________________________________________________________
Modifizieren / Löschen von Objekten:

  Die Codezeile aktivieren:
  - mit der Maus in die Zeile klicken  oder
  - mit den Cursortasten hinauf/hinunter

  Objektzeile im  Editfenster modifizieren oder löschen.

  Zu jeder Zeit kann man mit der Cursortaste hinauf/hinunter jeden
    Programmzustand wiederherstellen.

  Copy und Paste (Text kopieren / einfügen):
   Text selektieren (mit Maus oder ShiftTaste und CursorTasten);
   Selektierten Text kopieren: Ctrl-C     oder
   Selektierten Text kopieren und löschen: Ctrl-X
   Text einfügen (paste): Ctrl-V

  Zurück an das Programmende:
    - mit Taste Esc
    - oder mit dem "END" - Button
    - oder mit der Cursortaste im Editfenster ganz nach unten.


____________________________________________________________________________
Suchen von Objekten:
  
  Selektieren Sie das Objekt mit der rechten Maustaste, Funktion "Edit"
    (aktuelle Zeile im Editor muss leer sein);
  der Cursor wird im Editfenster auf die zum Objekt gehörige Textzeile
  positioniert.


____________________________________________________________________________
STEP - Button
  Code-Zeilen einzeln ausführen.


END - Button
  das Programm bis zum Dateiende abarbeiten.
  Mit der ESCAPE-Taste kann der Ablauf unterbrochen werden.
