activate function

only in mode CAD -
  select object with movable points (P,L,C,S)
   with right mousebutton, "edit curve"




move points of curve in plane

Activate checkbox "plane".
Definition of plane where point is moved.
  Default is the active constructionplane.
  Key into inputfield:
    "RZ" for X-Y-plane
    "RX" (for Y-Z-plane) or "RY" for X-Z-plane
    or select previously defined plane ("R#").
    or select plane in browser.



move points of curve along vector

Activate checkbox "vector".

Definition of a vector. Points will be moved along this vector.    
  Default is the normalvektor of the active plane ("DZ" eg for plane "RZ")
  or key into inputfield:
  - defaultvectors (DX,DY, DZ, DIX, DIX, oder DIZ) -
  - oder select vector in browser
  - or define plane (its normalvektor will be used)




move points of curve along vector with fixed length

  "vector"  like above
  "length"  key vectorlength





modify

move points with mouse

All movable points will be displayed as yellow rectangles.
Select a yellow rectangle -
Now all objects depending on this point will be modified dynamically.
Stop with left mousebutton.
Continue with "Save" or "Cancel".
 
 


delete

Delete points:

Only for curves with more than 3 points.
Select the point to be deleted.
Continue with next point or "Save" or "Cancel".




insert

Insert points:
Only for curves with more than 3 points.
Select the active curve at the position where to insert a point.
Continue with next point or "Save" or "Cancel".


Save / Cancel
 Cancel  - restore previous state; (also Esc)
 Save    - store point(s) temporary (also right mousebutton).



Exit

Exit CurveEditor with "Exit"-Button or right mousebutton.

Undo changes after exit with UNDO-icon.
