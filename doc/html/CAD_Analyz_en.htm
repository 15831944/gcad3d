Analysis Functions.




Analyze Point/Position    ANALYZP

Displays the point coordinates.
Select one vertex: Select /”PT--Vertex"





Analyze Object           ANALYZO

Analyze Items ANALYZO
Lists object type and coordinates.


See also menu Dump/Object (at top).

Analyze Distance PT-PT    ANALYZD1

Measures the distance between 2 points.


Select one vertex:  /”PT--Vertex"

Analyze Dist. PT-Obj   ANALYZD2

Measures the normal distance from point objects.
Currently only:
   Point/Line normal distance


   Point/Circle normal distance

Analyze Vertex         ANALYZV

Displays the position of the vertex points (the points of the 3D display)
  No analytical accuracy;
  The accuracy is dependent on the representation scale.
