Obj - Browser
 
 
 In the modes VWR and CAD the browser window is left of the graphics window 

  The first node is the main-model 
  all internal sub-models are presented below this. 
  
  All object-types of the active model are presented;

  - Select an object-type (left mouse-button) to display all objects of this type
    Reselect object-type to close this group.

  - Select an object to hilite object and add it to the active group;
    In CAD-mode the object-ID is written into the active input-field.
    Reselect object to unhilite object and remove it from the active group.

  Select a sub-model or the main-model with the right mouse-button to activate.
