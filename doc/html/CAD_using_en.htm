Creating objects: 

 You have to select a CAD function. Example: 
    "PT cartes" creates a point,
    "LN PT PT"  creates a line from point to point. 

  All required fields are now displayed.
  The entries are made by selecting, indicating or the keyboard.
  Indicate: activate the left mouse button at the appropriate position.
  Close your inputs with the Tab key.

  When all the necessary entries are made, the item is displayed. 
  All input values can still be changed,
  using the Tab key updates the display.
  Use the Esc key to clear a previously filled-in field. 

  The object will be stored and created with the OK button
  or with the Enter key ("Return")
  or with the right mouse button.

  When entering vectors, variables or modifiers,
  use the PageUp or PageDown buttons to select one of the existing values. 

  Cancel the active function:
  - select other CAD - function or
  - key esc or
  - select the Cancel - button



Modifying objects: 
  Finish the active CAD-function with the Esc-key.
    (several times if inputfields are not empty).

  Select the the geometric object with the right mousebutton,
  function "edit". The appropriate function is called and all parameters
    are entered in the input boxes.
  As an active CAD mode "MODIFY" is displayed.

  Modify the parameters (see "Creating objects" above).

  The modified object must be stored:
  - with the "OK-button
  - or with the right mousebutton.

  Go back to the end of the model, into CAD mode "ADD" (append):
  - with key Escape
  - or with the button "Cancel".



Deleting objects: 
  Finish the active CAD-function with the Esc-key.
    (several times if inputfields are not empty).


 To delete the most recently created object:
  Use Ctr-Z or the UNDO-button (red arrow to the left)
  Restoring objects: use Ctrl-Y or the REDO-button (green arrow to the right)


 To delete a particular object: 
  Select the geometric object with the right mousebutten,
  use function "Delete".

  The selected object and all dependent objects are deleted !


 Delete a group of objects:
  see "Gruppe"


 Remove all deleted objects permanent: 
  Use key "F4" (Modify/APT-Clean).



Searching for objects: 
  Use checkbox "Search/Name"
