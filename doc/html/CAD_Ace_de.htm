Active ConstructionPlane

Definition der ab nun gueltigen Konstruktionsebene -
  (eines Userkoordinatensytems, UCS).
  Zur Definition kann eine Standardebene oder eine zuvor definierte Ebene
    benutzt werden. (Standardebenen: RX oder RY oder RZ).
    (siehe CAD-PLN, Definition Ebenen)
  Der Nullpunkt der gewaehlten Ebene ist der neue Koordinatennullpunkt.
  Auf die Konstruktionsebene bezogen sind ab jetzt:
  - Koordinatenangaben
  - angezeigte Positionen 

  Die Anzeige der Mausposition kann relativ oder absolut erfolgen;
    relativ: (Default-) die Koordinaten im neuen Koordinatensystem;
    absolut: alle Koordinaten auf den absoluten Nullpunkt bezogen.
  Umschalten mit den Radiobuttons "rel" und "abs" neben der Anzeige
    der Mausposition.

  Ruecksetzen auf das absolute Koordinatensystem: Reset ConstrPlane

Beispiel:
R20=P(100 0 0) DZ DY
# Nullpunkt in 100,0,0; neue X-Achse in 0,10,0
R20
# Aktivieren als neues UserKoordinatensystem.
...

Siehe auch Beispielmodell sample_ucs1.gcad




  
Reset ConstrPlane

Ruecksetzen auf das absolute Koordinatensystem.

Ebene 0 definiert das absolute Koordinatensystem;
  es wird Ausgabecode "R0" erzeugt.
