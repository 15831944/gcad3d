Codes und Formate:

____________________________________________________________________________

- Definitionscodes  erzeugen geometrische Objekte,
                    das Format ist: {ObjektName} = {Parameter}
                    zB. "P20=P(100 0 0)"

- Bearbeitungscodes Zustandsänderungen, die sofort ausgeführt werden
                    bestehen aus einer direkten Anweisung;
                    zB "DEFCOL 0 0 220"    (Standardfarbe blau)
                     SHOW (hide) ATTL (Lineattributes) ATTS (Surfaceattributes)
                     MODE (Anzeige ..)

- Programmcodes     steuern den Ablauf der codes;
                    zB "PRI "P20=" P20"   (print point P20)
                     IF JUMP CALL EXIT DEBUG ..
