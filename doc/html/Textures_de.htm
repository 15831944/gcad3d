Texturen


Für Texturen können JPG- und BMP-Files verwendet werden.

Texturen können für 2D-Tags verwendet werden
Texturen können auf Surfaces aufgebracht werden ..


Texturen für 2D-Tags:  
  Ein Bild wird immer normal zum Betrachter dargestellt.
  Interaktive Funktion: CAD / N-Image
 
 
 
Texturen für Surfaces:
  Interaktive Funktion: Modify / SurfaceTexture.
  Texturen können auf folgende Surfaces aufgebracht werden:
  - planare Flächen
  - Meshes.
  Die Textur kann skaliert, verschoben und gedreht werden (modifizieren).
  Maßstab und Offset können für x und y getrennt definiert werden.
  Bearbeiten der Bilder fuer die Texturen zB mit Gimp:
    - Entzerren mit: Werkzeuge/Transformation/Perspektive
    - Zuschneiden mit: Werkzeuge/Transformation/Zuschneiden
    - Skalieren mit: Bild/Bild skalieren
      Fuer durchschnittliche Qualitaet: einen Meter auf ca 100 Pixel skalieren.





Load Texture from File
 Eine Textur aus einer Datei laden und aktivieren:
 Selektieren Sie eine JPG- oder BMP-Datei;
 der Texturname wird nun als aktive Textur angezeigt.


activate Texture from Surf
 Eine bereits verwendete Textur aktivieren:
 Selektieren sie eine Fläche mit Textur.
   (unhilite Textur (nur im Mode VWR): rechte Maustaste)
 Es werden nun Textur und Fläche aktiviert.
 Sie können nun die Textur modifizieren -
 oder auf eine andere Fläche aufbringen.


apply active Texture
 Die aktive Textur auf eine Fläche aufbringen.
 Selektieren Sie eine Surface.
 

remove Texture
 Textur von Fläche entfernen.
 Selektieren Sie eine Surface mit Textur.


fit Size
  Ein: das Bild wird auf die Grösse der Fläche skaliert
  Aus: das Bild wird in Originalgrösse auf die Fläche montiert


Reload
  Das Bild für die aktive Textur wird neu geladen.
  (wenn zB ein bereits geladenes Bild verändert wurde ..)







Scale - Offest - RotAng / Textur modifizieren:

- Scale X/Y  (Skalieren Textur; Cursortasten)
- Offset X/Y (Verschieben Textur; Cursortasten)
- RotAng (Verdrehwinkel in Grad; Cursor-Links/Rechts)

Veränderung der Werte über die Tastatur: Update mit der Return-Taste.
Veränderund der Werte mit den Cursortasten: Update erfolgt dynamisch.
Nächstes Eingabefeld mit der Tab-Taste.





Anhand eines Foto soll Masze analysiert werden:
  Das Foto entzerren.
  Eine Flaeche mit der genauen Groesse des Fotos machen;
    Wenn auf dem Foto eine Flaeche von 3 x 5 m dargestellt ist,
    soll eine Rechteckflaeche von 3x5 erzeugt werden (in Z=0)
      S20=REC P(0 0 0) D(5 0 0) D(0 3 0)
      A20=S20
  Textur applizieren.
  Punkte setzen (Select / PT -- Position)
  Punkte messen (Analyze Dist).
