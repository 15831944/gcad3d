Bedienung Funktion CAD:


  Hauptfunktion CAD.
  Interaktives Erzeugen / Modifizieren der Objektcodes.


 
Aktivieren der Funktion CAD:
   Die Checkbox CAD selektieren.

 
Button Help CAD:
  Hilfe zu CAD (diese Datei).


Button Help Func:
  Hilfe zur momentan aktiven CAD-Funktion;
    zB zur Funktion Punkte - "PT cartes"


Button Help Input:
  Hilfe zum momentan aktiven Eingabefeld
    (wie das erforderliche Eingabeobjekt erzeugt werden kann).




____________________________________________________________________________
Erzeugen von Objekten:
  Selektieren sie eine CAD-Funktion. Beispiel:
    "PT cartes"   erzeugt eine Punkt,
    "LN PT PT"    erzeugt eine Linie von Punkt zu Punkt.

  Es werden nun alle erforderlichen Eingabefelder angezeigt;
  die Eingaben erfolgen durch Selektion, Indicate  oder mit der Tastatur.
  Indicate: Anzeige einer Position mit der linken Maustaste.
  Schliessen Sie Tastatureingaben mit der Tab-Taste ab.

  Sind alle erforderlichen Eingaben gemacht, wird das Element angezeigt. 
  Alle Eingabewerte können noch verändert werden,
  mit der Tab-Taste wird die Anzeige sofort korrigiert.
  Mit der Esc-Taste wird das zuvor belegte Eingabefeld gelöscht.

  Mit dem OK-Button oder der Eingabetaste ("Return") oder mit der
  rechten Maustaste wird das Objekt gespeichert und erzeugt.

  Bei der Eingabe von Vektoren, Variablen oder Modifikatoren können Sie
  mit Hilfe der PageUp  oder PageDown - Tasten aus den vorhandenen Werten
  auswählen.

  Abbrechen einer Funktion:
  - andere CAD-Funktion selektieren oder
  - Esc - Taste oder
  - Button "Cancel"

 
 
Modifizieren von Objekten:
  Beenden Sie die aktive CAD-Funktion mit der Esc-Taste
    (mehrmals Esc wenn Eingabefelder nicht leer sind)

  Selektieren Sie das zu ändernde Objekt mit der rechten Maustaste,
    Funktion "Edit".  Es wird die erzeugende CAD-Funktion gerufen
    und alle erzeugenden Parameter in die Eingabeboxen eingetragen.
    Als aktiver CAD-Modus wird nun  "MODIFY" angezeigt.

  Modifizieren Sie die Parameter (siehe Erzeugen von Objekten, weiter oben).

  Das modifizierte Objekt muss gespeichert werden:
    - mit dem "OK"-Button 
    - oder mit der rechten Maustaste

  Zurück an das Programmende, in den CAD-Modus "CREATE" (hintanfügen):
    - mit Taste Esc
    - oder mit Button "Cancel" 

 

Löschen von Objekten:
  Beenden Sie die aktive CAD-Funktion mit der Esc-Taste
    (mehrmals Esc wenn Eingabefelder nicht leer sind)


 Das zuletzt erzeugte Objekt löschen:
   Ctr-Z oder Undo-Button (roter Pfeil nach links)
   Wiederherstellen: Ctrl-Y oder Redo-Button  (grüner Pfeil nach rechts).


 Ein bestimmtes Objekt löschen:
  Selektieren Sie das zu ändernde Objekt mit der rechten Maustaste,
    Funktion "Löschen".

  Es werden auch alle vom selektierten Objekt abhängigen Objekte gelöscht !


 Eine Gruppe von Objekten löschen:
  Siehe "Gruppe"


 Entfernen aller gelöschten Zeilen:
   Taste "F4"   (Modify / APT-Clean).

 

Suchen von Objekten:
  Mit Checkbox "Search/Name"
 
 
Zusatzfunktionen zur Selektion von Objekten:
 Im Menü "Select" gibt es folgende Hilfsfunktionen:
  get position of cursor on construction-plane
  get position of cursor on selected object
  get parametric position of cursor on selected object
  get parentObject
