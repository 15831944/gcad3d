Funktionen Menü Select:



get point = position of cursor on construction-plane
  Bei der nachfolgenden Selektion wird der Punkt auf der
  Konstruktionsebene geliefert.


get point = (parametric) position of cursor on object
  Bei der nachfolgenden Selektion wird der (parametrische) Punkt
  auf dem selektierten Objekt geliefert.

  Wird z.B. der Startpunkt einer Linie selektiert, ist das Ergebnis:
  P(L20 MOD(1))


get line = segment of polygon / contour
  Bei der nachfolgenden Selektion wird eine Linie geliefert.
  Die Linie wird aus dem selektierten Polygon oder der Kontur extrahiert.

  Das 2. Teilsegment eines Polygons:
  L(S20 MOD(2))






Eine Gruppe von Objekten selektieren:
  den Ctrl-key und die linke Maus-Taste drücken und gedrückt halten,
  die Maus verschieben ..
