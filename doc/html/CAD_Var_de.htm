CAD-Funktionen Variable:



siehe auch Codes Variable

Variable benutzen den Kennbuchstaben V.

Implizite Definition durch die Funktion VAL().
  Beispiel: VAL(P20 P21)  - ergibt den Abstand zwischen 2 Punkten.

Variable werden temporär als horizontale Linie angezeigt;
  die Länge entspricht dem Wert.



____________________________________________________________________________


  
V Variable - Value

- Direktzuweisung einer Variablen

- mit Pageup / PageDown-Taste eine definierte Variable auswählen.





  
V X/Y/Z-part PT|VC

Den X- oder Y- oder Z-Anteil eines Punktes oder Vektors kopieren.

# die X-Koordinate eines Punktes kopieren
V20 = X(P20)






  
V PT-PT Dist.[Direction]

 Variable = Länge zwischen 2 Punkten [in Richtung]

[Direction]   optional; keine Angabe: direkte Länge zwischen 2 Punkten
              Abstand entlang eines Vektors: Definition eines Vektors

Beispiel:
V20 = P20 P21
# Differenzabstand von Punkt P20 bis P21 in Richtung Z:
V21=P20 P21 DZ





  
V LN      length

 Variable = Länge einer Strecke

V20 = L20





  
V PT - LN Perp.Dist

 Variable = Normalabstand eines Punktes von einer Strecke

V20 = P20 L20





  
V CIR     Radius

 Variable = Radius eines Kreises
 
V20 = C20
