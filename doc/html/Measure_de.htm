Measure


Taste Esc löscht die zuletzt gemachte Eingabe und setzt Fehlermeldung zuück.



Point/Point   (distance)


Den Abstand zwischen 2 Punkten messen.

Auf subModellen können keine Punkte selektiert werden;
  Sie können aber alle Darstellungspunkte benutzen -
  (schalten Sie dazu die Checkbox "Shade" aus).




Point/Object  (normal distance)


Den Normalabstand von Punkt zu Objekt messen.
Objekt: Linie, Kreis, Kurve.
